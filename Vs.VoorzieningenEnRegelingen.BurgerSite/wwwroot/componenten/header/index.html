Toegankelijkheid

Voor de toegankelijkheid van de link met het logo is het niet voldoende om alleen een title op te nemen, omdat screenreaders dat niet altijd goed verwerken.
Neem daarom naast de title ook altijd een tekst in de link op.
Omdat we die tekst niet zichtbaar willen hebben, maar er alleen voor screenreaders is, geven we die dan de class screenreader-only. Natuurlijk moet de link tekst een duidelijke beschrijving bevatten.

Standaard header

Gebruik de standaard header wanneer er ook een navigatiebalk aanwezig is.

dienst uitvoering onderwijs, naar de homepage

Transparante header

De transparante header is bedoeld voor pagina’s waarin geen navigatiebalk zichtbaar is.

dienst uitvoering onderwijs, naar de homepage

Witte header

De witte header is bedoeld voor header’s waar een lichte achtergrondkleur (of afbeelding) de tekst onleesbaar maakt.

dienst uitvoering onderwijs, naar de homepage
