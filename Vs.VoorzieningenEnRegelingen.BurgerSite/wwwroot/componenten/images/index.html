Afbeeldingen

Voor afbeeldingen kan de standaard img tag gebruikt worden. Voorzie de tag altijd van een alt attribuut, zodat visueel beperkten ook kunnen achterhalen waar de afbeelding voor dient.
Geef daarnaast bij voorkeur de width mee, zodat een browser alvast ruimte kan reseveren voor de afbeelding.
